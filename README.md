# Sudoku Solver 🧩

## Overview

This project is a Sudoku Solver developed in Python. It automatically solves an incomplete Sudoku puzzle using the Backtracking Algorithm. The program takes a Sudoku grid as input, finds empty cells, and fills them with valid numbers while following Sudoku rules.

## Features

* Solves Sudoku puzzles automatically
* Uses the Backtracking Algorithm
* Validates rows, columns, and 3×3 subgrids
* Displays the solved Sudoku puzzle
* Beginner-friendly Python project

## Technologies Used

* Python 3

## How It Works

1. The program scans the Sudoku grid for an empty cell.
2. It tries numbers from 1 to 9.
3. Checks whether the number is valid according to Sudoku rules.
4. If valid, it places the number and moves to the next empty cell.
5. If no valid number is found, it backtracks and tries a different number.
6. The process continues until the puzzle is solved.
